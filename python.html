Introducción

Python es un lenguaje de programación fácil de aprender, con una sintaxis clara y elegante. Es utilizado en muchas áreas, como el desarrollo web, análisis de datos, automatización de tareas, inteligencia artificial y más.

Variables y tipos de datos

Una variable es un espacio de memoria donde guardamos información para usarla después. En Python, no necesitas indicar el tipo, el lenguaje lo detecta automáticamente.


nombre = "Ana"       # Texto (str)
edad = 25            # Número entero (int)
pi = 3.1416          # Número decimal (float)
es_estudiante = True # Valor booleano (bool)

Tipos de datos básicos:

  • str: Cadenas de texto. Ejemplo: "Hola"
  • nt: Números enteros. Ejemplo: 10
  • float: Números con decimales. Ejemplo: 3.14
  • bool: Valores booleanos. True o False

Estruccturas de control

Son instrucciones que permiten tomar decisiones o repetir acciones según condiciones.

Condicional if

Sirve para ejecutar código solo si se cumple una condición.


edad = 18
if edad >= 18:
    print("Eres mayor de edad")
else:
    print("Eres menor de edad")

Bucle for

Repite una acción un número determinado de veces.


for i in range(5):  # Desde 0 hasta 4
  print(i)

Bucle while

Repite una acción mientras se cumpla una condición.


contador = 0
while contador < 5:
  print(contador)
  contador += 1

Funciones

Una función es un bloque de código que se puede reutilizar. Se puede ejecutar cuando se necesite.


def saludar(nombre):
  print(f"Hola, {nombre}")

saludar("Carlos")

def: palabra clave para definir una función.

nombre: es un parámetro, un dato que se le pasa a la función.

Listas y diccionarios

Listas

Una lista guarda varios valores en una sola variable.


frutas = ["manzana", "banana", "uva"]
print(frutas[0])         # manzana (índice 0)
frutas.append("naranja") # Agrega al final

Diccionarios

Un diccionario guarda datos en forma de clave:valor, como una agenda o ficha.


persona = {"nombre": "Luis", "edad": 30}
print(persona["nombre"])  # Luis
persona["edad"] = 31      # Actualiza el valor

Módulos

Un módulo es un archivo con funciones que puedes usar en otros programas. Python ya trae muchos módulos útiles.


import math
print(math.sqrt(16))  # Raíz cuadrada de 16 → 4.0

Entradas y salidas

Entrada input(): sirve para pedirle datos al usuario.


nombre = input("¿Cómo te llamas? ")
print(f"Hola, {nombre}")
